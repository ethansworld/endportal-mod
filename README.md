# End Portal Spawner Mod

A Minecraft 1.20.1 Forge mod that adds a staff with infinite uses which spawns a 3x3 End Portal wherever you use it.

## Features
- Adds an "End Portal Staff" with infinite durability and uses
- Spawns a 3x3 End Portal when used on a block
- Custom item icon for the staff

## Getting Started
1. **Build:**  
   Run `./gradlew build`  
   The output JAR is in `build/libs/`
2. **Install:**  
   Place the jar file in your Minecraft 1.20.1 `mods` folder (Forge required)
